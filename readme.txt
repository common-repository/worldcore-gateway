Payment gateway plugin to accept Worldcore on your Woocommerce based web-site

== Description ==

Worldcore is an innovative Internet-based financial services provider.

Worldcore online payment service provider offers a wide range of global payment 
solutions including access to bank payments, e-currency payments, prepaid debit 
cards, mass payments, payouts to any Visa/Mastercard credit/debit cards and 
international payment processing for both individuals and businesses.

== Key features ==

* Instant payments
* Automatic payment confirmations
* Support of Bank wire transfer payment method

== Installation ==

1. Login WordPress Admin Control panel
2. Go to "Plugins" > "Add new" and click on "Upload plugin" to install Worldcore payment method for WooCommerce
3. Go to "Plugins" > "Installed Plugins" to activate Worldcore payment plugin
4. Go to "WooCommerce" > "Settings" > "Checkout" > "Worldcore" and enter Wordcore API configuration credetials (Account ID, API key and API password) 
5. Enter the following Status URL under API settings section of Worldcore Member's Area: https://yoursite.com/wc_confirmation
